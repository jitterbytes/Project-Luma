# Project-Luma

**Luma** is an interactive ambient pixel object designed for the desk - a small, button-driven device that displays calm, expressive pixel visuals using an LED matrix.

<img src="img/Landscape_coverpage.png" width="1000">

## What is Luma?
Luma is not a utility tool, it does not give you information...Its an ambient desk object that is just meant to be watched, interacted with, and enjoyed.  
It features a simple **menu system** and multiple pixel-based visual interactions, each responding directly to the user input.  

## Current Features 
- WS2812B 8x8 Matrix
- Button driven menu navigation
- Interactive ambient animations
- Multiple interaction modes

## Hardware Overview
- **MCU** ESP32 C3 Supermini Board
- **Display** WS2812B 8x8 Matrix
- **Inputs** 2 Tactile Buttons

## Firmware & Source Code
